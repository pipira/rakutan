## rakutanとは
一言で言うと、学内の生徒が授業の評価を投稿、閲覧できるサイトです。

### UVP（独自の価値提案 ≒ 一押しポイント）
あとで考える

- Topページ
    - 検索ボックス

- 授業一覧
    - 検索ボックス
        - 授業名
        - 教師名
        - 曜日
    - 
- 授業詳細
- 投稿一覧

- ログインフォーム
